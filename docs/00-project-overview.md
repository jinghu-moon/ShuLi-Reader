# 00 - 项目概述

## 项目名称

- **中文名**：书里
- **英文名**：InTale
- **包名**：com.shuli.reader

## 项目定位

沉浸式本地小说阅读器，专注于 TXT/EPUB 格式的高性能阅读体验。

## 目标用户

小说重度用户，需要：
- 大文件（100MB+）流畅阅读
- 多种翻页方式
- 个性化主题定制
- 多端同步阅读进度

## 核心价值

- **轻量**：遵循奥卡姆剃刀原则，最小化依赖
- **高性能**：极致性能优化，毫秒级渲染
- **沉浸**：专注阅读体验，去除干扰

## 技术栈

- 语言：Kotlin
- UI 框架：Jetpack Compose
- 最低版本：Android 12 (API 31)
- 开发方式：纯原生
