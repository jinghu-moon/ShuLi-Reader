“同学们，刚才这段话是文章的环境描写，在写作中环境描写必不可少，不仅能起到烘托氛围的作用，也能折射出主人公内心的情感，同学们听明白了吗？”

安静的教室里，所有人均正经端坐在位置上，聚精会神的听着语文 老师

韩晶柔媚悦耳的声音。我自然也不例外，挺直身躯目不转睛的看着韩老师优美动人的身姿，眼中流露出浓浓的喜爱。

我叫叶小天，今年13岁，正在念初一，由于妈妈去世的早所以一直与爸爸和大姨生活在一起。

爸爸学历不高只有大专文凭，为了能让我有个好的学习环境，同时也为了能够多挣点钱，爸爸一直在沿海的发达城市打工，一年之中几乎只有过年时才会回到家中。
