# 12 - 阅读界面设计

> 书里阅读器阅读界面的完整元素设计，涵盖页眉页脚、工具栏、点击热区、浮层弹窗与数据流关系。

## 设计目标

- **沉浸**：默认隐藏所有控件，最大化正文展示区域
- **高效**：单手可完成翻页、调节、跳转等核心操作
- **一致**：遵循 Legado 验证过的交互模式，降低用户学习成本

### 3.2 电池图标实现

```kotlin
// BatteryMonitor.kt
class BatteryMonitor(private val context: Context) {
    private val batteryLevel: StateFlow<Int> = ...

    fun getBatteryLevel(): Int {
        val batteryManager = context.getSystemService(Context.BATTERY_SERVICE) as BatteryManager
        return batteryManager.getIntProperty(BatteryManager.BATTERY_PROPERTY_CAPACITY)
    }

    fun isCharging(): Boolean {
        val intentFilter = IntentFilter(Intent.ACTION_BATTERY_CHANGED)
        val batteryStatus = context.registerReceiver(null, intentFilter)
        val status = batteryStatus?.getIntExtra(BatteryManager.EXTRA_STATUS, -1)
        return status == BatteryManager.BATTERY_STATUS_CHARGING
            || status == BatteryManager.BATTERY_STATUS_FULL
    }
}
```

## 六、点击热区划分

### 6.1 九宫格布局

```
┌─────────┬─────────┬─────────┐
│   TL    │   TC    │   TR    │  ← 上部 1/3
│  上一页 │  菜单   │  下一页 │
├─────────┼─────────┼─────────┤
│   ML    │   MC    │   MR    │  ← 中部 1/3
│  上一页 │  隐藏   │  下一页 │
├─────────┼─────────┼─────────┤
│   BL    │   BC    │   BR    │  ← 下部 1/3
│  上一页 │  工具栏 │  下一页 │
└─────────┴─────────┴─────────┘
```

### 6.2 热区行为定义

| 区域 | 单击 | 长按 |
|------|------|------|
| TL (左上) | 上一页 | - |
| TC (上中) | 显示工具栏 | - |
| TR (右上) | 下一页 | - |
| ML (左中) | 上一页 | - |
| MC (中中) | 显示/隐藏工具栏 | - |
| MR (右中) | 下一页 | - |
| BL (左下) | 上一页 | - |
| BC (下中) | 显示工具栏 | - |
| BR (右下) | 下一页 | - |

### 6.3 翻页方向适配

| 设置 | TL/TR | ML/MR | BL/BR |
|------|-------|-------|-------|
| 左右滑动 | 上/下一页 | 上/下一页 | 上/下一页 |
| 上下滚动 | 上/下一页 | 上/下一页 | 上/下一页 |

> 注：上下滚动模式下，翻页由滚动手势触发，点击热区仅在非滚动模式下生效。

## 十、与分页引擎的交互

### 10.1 页面数据流转

```
ChapterProvider.paginate(chapterText)
    │
    ▼
List<TextPage>
    │
    ├─ TextPage[0]: startCharOffset=0, endCharOffset=1200
    ├─ TextPage[1]: startCharOffset=1200, endCharOffset=2400
    └─ TextPage[2]: startCharOffset=2400, endCharOffset=3600
    │
    ▼
ReaderViewModel 持有当前 TextPage
    │
    ▼
ReaderScreen 渲染 TextPage.textLines
```

### 10.2 翻页数据流

```
用户滑动翻页
    │
    ▼
PageDelegate.onTouch() 处理手势
    │
    ▼
PageFactory.moveToNext() / moveToPrev()
    │
    ├─ 更新 currentPage
    ├─ 更新 pageIndex
    ├─ 计算 readProgress
    ├─ 保存阅读位置（防抖 500ms）
    │
    ▼
ReaderScreen recomposition → Canvas 重绘
```

## 十一、Canvas 绘制区域分配

```
┌─────────────────────────────────────────────────────────┐
│  paddingTop (24dp)                                      │
│  ┌─────────────────────────────────────────────────┐   │
│  │  页眉区域 (24dp)                                 │   │

│  │  │                                         │   │   │
│  │  │         正文渲染区域                     │   │   │
│  │  │     (visibleWidth × visibleHeight)      │   │   │
│  │  │                                         │   │   │
│  │  └─────────────────────────────────────────┘   │   │
│  │                                                 │   │
│  ├─────────────────────────────────────────────────┤   │
│  │  页脚区域 (24dp)                                 │   │

│                                                         │

可见区域计算：
visibleWidth = viewWidth - paddingLeft - paddingRight
visibleHeight = viewHeight - paddingTop - paddingBottom - headerHeight - footerHeight
```
