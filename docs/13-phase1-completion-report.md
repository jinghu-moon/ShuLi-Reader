## 核心功能实现

### 1. 文件解析系统
- **TXT 解析**: 支持元数据快速解析、mmap 流式读取、章节检测增强
- **EPUB 解析**: 支持元数据提取、目录解析、正文提取、安全防护
- **编码检测**: 支持 UTF-8、GBK 等多种编码自动检测

### 2. 阅读引擎
- **分页器**: 实现纯逻辑分页，支持段落、标题、缩进
- **缓存系统**: 实现 LRU 缓存，支持页面、章节多级缓存
- **虚拟化加载**: 实现章节预加载，只加载当前及前后章节

### 3. 渲染系统
- **Canvas 视图**: 实现背景、正文、页眉、页脚、进度条绘制
- **主题支持**: 支持浅色、暗色、纸质主题
- **离屏缓冲**: 实现页面预渲染到位图

### 4. 交互系统
- **热区计算**: 实现九宫格热区映射，支持翻页模式和滚动模式
- **状态机**: 实现工具栏、目录、设置等浮层状态管理
- **进度保存**: 实现防抖保存和立即保存

### 5. 数据模型
- **阅读位置**: 实现字符偏移作为权威位置
- **书签笔记**: 支持章节索引、字符偏移、选中文本
- **数据库迁移**: 实现版本化迁移，保护用户数据
