# 10 - 设置页面设计

> 书里阅读器设置页面的完整设计方案，涵盖所有配置项、交互逻辑与实现优先级。

## 设计原则

- **渐进式披露**：常用设置前置，高级设置折叠
- **即时预览**：字体、主题等变更实时生效
- **分组清晰**：按功能域划分，避免长列表
- **默认合理**：开箱即用，无需大量配置

## 十、页面结构

```
设置页面 (SettingsScreen)
├── 阅读器显示偏好
│   ├── 字体与排版 → FontSettingsScreen
│   ├── 页面布局 → LayoutSettingsScreen
│   └── 显示元素 → DisplayElementsScreen
├── 主题与外观
│   ├── 预设主题选择 → 主题卡片网格
│   ├── 自定义主题 → CustomThemeScreen
│   └── 自动切换 → AutoThemeSettings
├── 书库与导入
│   ├── 书库管理
│   ├── 存储管理 → StorageScreen
│   └── 导入设置
├── 阅读统计 → StatsScreen
├── 同步设置 → SyncScreen
├── 朗读设置 → TTSSettingsScreen
├── 高级设置
│   ├── 性能设置
│   ├── 实验性功能
│   └── 调试选项
└── 关于与版权 → AboutScreen
```
